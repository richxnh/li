li's framework,一个简单轻量的java web开发框架.

li,这是一个简单轻量的javaweb开发框架,包含了Mvc Dao ActiveRecord Ioc Aop等功能的极其小巧的实现.

Action代码示例

@Bean //注解一个类使之成为一个Ioc Bean
public class PeopleAction extends AbstractAction {
    @Inject //Ioc注入属性
    People peopleDao;

    @At("people_list.do") //注解一个Action方法
    public void list(Page page) { //方法参数适配
        setRequest("list", peopleDao.list(page));
        setRequest("page", page);
        view("people/list"); //展现视图
    }
}

ActiveRecord代码示例

@Bean
@Table("t_account") //注解这个类对应的数据表
public class Account extends Record<Account> { //数据对象继承于Record
    public Boolean login(Account account) {
        String sql = "WHERE username=#username AND password=#password";
        return null != find(sql, account); //利用超类中的方法
    }
}
